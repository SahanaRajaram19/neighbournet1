# Image Upload Feature - Complete Guide

## ✅ Image Upload System Implementation

You can now add pictures to all items in the NeighbourNet platform!

## 📸 Where to Upload Images:

### 1. **Admin Dashboard - Admin Items Page**
- **URL:** http://localhost/neighbournet1/admin-items.html
- **How:** Click "Add New Item" button → Upload image in modal form
- Features:
  - Drag & drop image upload
  - Click to select image
  - Image preview before saving
  - Supports JPG, PNG, GIF, WebP
  - Maximum 5MB file size

### 2. **Item Listing Pages**
- **URLs:** 
  - http://localhost/neighbournet1/borrow.html
  - http://localhost/neighbournet1/rent.html
  - http://localhost/neighbournet1/list-item.html
- Features:
  - Upload main item image
  - Support for multiple photos
  - Image preview on card
  - Auto-resize images

### 3. **Item Detail Page**
- **URL:** http://localhost/neighbournet1/item-detail.html
- Shows uploaded item images
- Large image view with details

### 4. **Home Page**
- **URL:** http://localhost/neighbournet1/index.html
- Featured items display with images
- Automatic image loading from database

## 🔧 Technical Details:

### Backend Files:
- **upload-image.php** - Handles image uploads
- **add-item.php** - Updated to accept image_url
- **migrate-db.php** - Adds image_url column to items table

### Upload Directory:
```
C:\xampp\htdocs\neighbournet1\uploads\items\
```
All uploaded images stored here.

### Database:
- **items table** - Now has `image_url` column
- Stores image path for each item
- If no image: Uses placeholder (auto-generated by category)

## 📋 Image Upload Flow:

1. **Select/Drag Image** → User picks image file
2. **Validate** → Check file type and size
3. **Upload** → Send to upload-image.php
4. **Save** → Get image URL back
5. **Database** → Save image URL with item data
6. **Display** → Show on all pages with items
