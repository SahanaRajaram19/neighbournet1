# 📸 Image Upload Feature - Quick Start

## ✅ What's New:

All pages now support **item pictures**! You can:

### 📤 Upload Images to Items:

1. **Admin Panel** → admin-items.html
   - Click "Add New Item"
   - Drag/drop or click to upload image
   - Image shows in admin table
   - Maximum 5MB, JPG/PNG/GIF/WebP

2. **All Item Pages** Show Images:
   - ✅ Home page - Featured items with pictures
   - ✅ Borrow page - Item list with thumbnails
   - ✅ Rent page - Item browsing with images
   - ✅ Item detail - Full image display
   - ✅ Admin dashboard - All images in table

3. **Image Features:**
   - Drag & drop upload
   - Click to browse
   - Preview before saving
   - Auto-sized for each page
   - Professional placeholders if no image

## 🎯 Try It Now:

### Steps:
1. Go to: **http://localhost/neighbournet1/admin-items.html**
2. Click: **Add New Item** button
3. Fill in:
   - Title: "My Power Drill"
   - Category: "Tools"
   - Price: "50"
   - Condition: "Excellent"
4. Upload Image:
   - Click the image upload area
   - Drag image OR click to browse
   - Select image from computer
   - See preview appear
5. Click: **Save Item**
6. View your item with image on:
   - http://localhost/neighbournet1/index.html (Featured)
   - http://localhost/neighbournet1/borrow.html (Full list)
   - http://localhost/neighbournet1/admin-items.html (Admin table)
